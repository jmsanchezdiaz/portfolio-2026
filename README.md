# Portfolio

Personal portfolio built with React 19, TypeScript and Vite.

## Tech stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Lucide React

## Features

- Bilingual (EN / ES)
- Sections: Hero, About, Experience, Education, Skills, Contact
- Smooth animations with Framer Motion

## Getting started

```bash
pnpm install
pnpm dev
```

## Commands

```bash
pnpm dev       # Start dev server with HMR
pnpm build     # Type-check then bundle
pnpm lint      # Run ESLint
pnpm preview   # Preview production build
```
